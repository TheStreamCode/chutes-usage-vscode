# Support

Use GitHub Issues for bug reports, feature requests, and support questions:

- `https://github.com/TheStreamCode/chutes-usage-vscode/issues`

## Before Opening An Issue

Check the public documentation first:

- `README.md`
- `docs/user-guide.md`
- `docs/troubleshooting.md`

## What To Include

Please include:

- extension version
- VS Code version
- operating system
- steps to reproduce
- expected behavior
- actual behavior

If the problem is related to API parsing or quota behavior, include a redacted payload when possible.
